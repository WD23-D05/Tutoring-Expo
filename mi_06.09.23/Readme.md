Wenn zwei Argumente übergeben wurden, lesen Sie sie aus process.argv und konvertieren Sie sie in Ganzzahlen (Integer).

Geben Sie das Ergebnis auf der Konsole aus.

Beispiel:   $ node main.js 5 7
